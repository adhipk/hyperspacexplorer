Server Contract

Serve ordinary HTML with a Hyperclay-compatible save endpoint.

The preferred host is Hyperclay Local's server core without the Electron shell. This repo's server documents the compatibility contract: serve local files, provide the Hyperclay runtime, accept Hyperclay save requests, and write clean HTML back to disk.

Save request

POST /_/save
Page-URL: http://localhost:5173/index.html
Content-Type: text/plain

<!DOCTYPE html>...

Responsibilities

  1. Set local owner cookies for Hyperclay edit mode.
  2. Serve HTML, CSS, JavaScript, SVG, and other project assets.
  3. Provide HyperclayJS and the Hyperspace runtime through source includes or response-time injection.
  4. Resolve save targets from the Page-URL header.
  5. Allow writes only to safe .html and .htmlclay paths.
  6. Create backups before overwriting existing files.
  7. Strip runtime UI before persisting saved HTML.
