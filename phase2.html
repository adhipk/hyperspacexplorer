Server Contract

Serve ordinary HTML with a Hyperclay-compatible save endpoint.

The server is intentionally small. It serves local project files, injects the HyperclayJS and Hyperspace runtime only at response time, accepts Hyperclay save requests, and writes the clean HTML file back to disk.

Save request

POST /_/save
Page-URL: http://localhost:5173/index.html
Content-Type: text/plain

<!DOCTYPE html>...

Responsibilities

  1. Set local owner cookies for Hyperclay edit mode.
  2. Serve HTML, CSS, JavaScript, SVG, and other project assets.
  3. Inject runtime scripts into HTML responses, not source files.
  4. Resolve save targets from the Page-URL header.
  5. Allow writes only to safe .html and .htmlclay paths.
  6. Create backups before overwriting existing files.
  7. Strip runtime UI before persisting saved HTML.
