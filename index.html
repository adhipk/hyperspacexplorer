Project Documentation

HTML documents that can be marked up in place.

Hyperspace treats an HTML file as the readable source of truth. The page can contain editable regions, interactive form-like controls, and nearby comments that remain meaningful in raw HTML. Edit is good

Runtime split

Hyperclay core

HyperclayJS owns edit mode, save snapshots, persistence helpers, dialogs, DOM helpers, and the /_/save lifecycle.

Hyperspace layer

Hyperspace adds a small toolbar, comment placement, styling, and AI-facing document conventions.

Local host

A Hyperclay-compatible local server serves the files, injects runtime scripts, and writes saved HTML back to disk.

Basic comment behavior

Select the comment tool, click where the note belongs, type the comment, and click away. The comment is inserted into the closest document container and can float visually through CSS position state.

Source convention
